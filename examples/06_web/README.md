# Module 1: Web search

Give one Responses request access to hosted web search. Search calls and
citations return inside the complete response object.

## Run

```bash
./scripts/run.sh examples/06_web/example.py
./scripts/run.sh examples/06_web/nicer_example.py
```

`example.py` uses the OpenAI SDK directly. `nicer_example.py` uses `lab_llm`.
On Windows, use `.\scripts\run.ps1` and backslashes.

[OpenAI: web search](https://developers.openai.com/api/docs/guides/tools-web-search)
