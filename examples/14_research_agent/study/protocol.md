# Analysis protocol

Protocol version: 2.1

Four participants are enrolled: P001 through P004. Each should have one usable
interview transcript.

The anxiety-frequency item uses a 1–5 scale:

1. Never
2. Rarely
3. Sometimes
4. Often
5. Nearly every day

The planned analysis uses model `gpt-5.4` and prompt version `2.0`. Ratings
outside the documented scale must be treated as invalid, not clipped.
