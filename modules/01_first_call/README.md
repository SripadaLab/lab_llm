# Module 1 — From ChatGPT to programmatic tools

The first programmatic call: send input from code, get a model response back.

**Take-home:** the `call_llm` helper (in `lab_llm/calls.py`) — a clean starting
point for prompting from Python instead of the ChatGPT UI.

## Run

```bash
python modules/01_first_call/example.py
```

## What to expect

A two-sentence summary printed to the terminal, followed by a `Usage(...)` line
with token counts.
